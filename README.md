## Installation

```bash
$ npm install
```


## Setup
- Rename <code>.env.example</code> to <code>.env</code>

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Notes
You need to manually import trip data to database
[Grab trip data from this url](https://github.com/DataTalksClub/nyc-tlc-data/releases/tag/yellow)
